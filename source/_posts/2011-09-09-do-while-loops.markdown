``` java
class apples{
  public static void main (String[] args) {
    int counter = 0;

    do {
    	System.out.println(counter);
    	counter++;
    } while(counter <= 10);

  }
}
```

Do something at at least, use a do loop, anytime you test something and is
true it runs it.
