Change Back Windows 8 Start Menu Back

Sep 14th, 2011

The new start menu when you you click on the Windows logo isn’t the same anymore in Windows 8, but now you can change it back to the original design by changing the registry key.

Change the value of RPEnabled from “1” to “0” in HKEY_CURRENT_USER\Software\Microsoft\Windows\CurrentVersion\Explorer

Redirect Tumblr to Anything

Sep 11th, 2011

I have a Tumblr blog which is my primary one but don’t use it, I thought I would just redirect it to google.com, did a quick search and found something on StackExchange. It is a simple Javascript if statement.

<script type="text/javascript">
if(location.href == http://www.youraddress.tumblr.com/) location.replace(http://www.google.com);;
</script>
