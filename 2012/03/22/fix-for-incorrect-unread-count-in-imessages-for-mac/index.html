Fix for Incorrect Unread Count in iMessages for Mac

Mar 22nd, 2012

There is currently a bug on iMessages for Mac which shows incorrect count of unread messages, this bug comes and goes. There is a temporary fix while Apple corrects this with the next update (which is probably not soon).

  1. Open Terminal.
  2. Enter “killall Dock” (without the quote marks).
